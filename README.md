# Cryptocurrency Transaction Classification using Graph Neural Networks

This project uses Graph Neural Networks (GNN) to classify cryptocurrency transactions using the Elliptic dataset.

## Technologies
- Python
- PyTorch
- Pandas
- NumPy
- Scikit-learn
- Torch Geometric

## Features
- Data preprocessing and feature engineering
- Graph construction from transaction data
- Graph Neural Network model implementation
- Model evaluation using classification metrics
- ROC and Precision-Recall curve visualization

## Dataset
Elliptic Bitcoin Dataset

## Results
Model performance evaluated using:
- Accuracy
- Confusion Matrix
- ROC Curve
- Precision Recall Curve
